Política de Privacidad

Este bot almacena la siguiente información:

  • ID de usuario de Discord
  • Nombre de usuario (display name)
  • Puntos asignados dentro del servidor

Los datos se almacenan localmente en un archivo JSON y no se comparten con terceros.

Podés solicitar la eliminación de tus datos contactando al administrador del servidor o usando el comando /delpoints.

No recolectamos mensajes, contenido de canales ni información personal sensible.
